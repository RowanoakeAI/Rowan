Introduction

Welcome to the Rowan documentation. Rowan is a friendly AI assistant designed to help you with various tasks and make your life easier.

Getting Started

To get started with Rowan, follow these steps:

  1. Clone the repository: git clone https://github.com/RowanoakeAI/Rowan.git
  2. Install dependencies: pip install -r requirements.txt
  3. Set up environment variables: cp config/.env.example config/.env
