Introduction

The Rowan API allows you to interact with the Rowan assistant programmatically. You can send messages, manage tasks, and more.

Authentication

To use the API, you need to include an API key in your requests. The API key should be included in the X-API-Key header.

Endpoints

GET /status

Check the status of the Rowan API.

GET /status

POST /chat

Send a message to Rowan and receive a response.

POST /chat
{
    "message": "Hello, Rowan!",
    "context_type": "casual",
    "source": "api"
}

Examples

Here are some examples of how to use the Rowan API.

Python

import requests

api_key = 'your_api_key'
url = 'http://localhost:7692/chat'
headers = {'X-API-Key': api_key}
data = {
    'message': 'Hello, Rowan!',
    'context_type': 'casual',
    'source': 'api'
}

response = requests.post(url, json=data, headers=headers)
print(response.json())

Error Codes

The following error codes may be returned by the API:

  • 401 Unauthorized: Invalid API key.
  • 429 Too Many Requests: Rate limit exceeded.
  • 500 Internal Server Error: An error occurred on the server.
